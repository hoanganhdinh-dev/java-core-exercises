# Java English Vocabulary

This folder contains English vocabulary collected while learning Java.

## Weeks

### Week 1 - Java Core Basics

| English | Meaning | Example |
|----------|---------|---------|
| Variable | Biến | Store data in memory |
| Data Type | Kiểu dữ liệu | int, double, boolean |
| Primitive Type | Kiểu dữ liệu nguyên thủy | int, char, boolean |
| Reference Type | Kiểu tham chiếu | String, Object |
| Declaration | Khai báo | Declare a variable |
| Initialization | Khởi tạo | Initialize a variable |
| Assignment | Gán giá trị | Assign value to variable |
| Operator | Toán tử | Arithmetic operator |
| Arithmetic Operator | Toán tử số học | + - * / % |
| Comparison Operator | Toán tử so sánh | > < == != |
| Logical Operator | Toán tử logic | && || ! |
| Condition | Điều kiện | if condition |
| Statement | Câu lệnh | Java statement |
| Expression | Biểu thức | a + b |
| Loop | Vòng lặp | for loop, while loop |
| Iteration | Lần lặp | Each iteration |
| Method | Phương thức | Call a method |
| Parameter | Tham số | Method parameter |
| Argument | Đối số | Pass argument |
| Return Value | Giá trị trả về | Return result |
| Scope | Phạm vi biến | Variable scope |
| Constant | Hằng số | final variable |
| Exception | Ngoại lệ | Handle exception |
| Input | Dữ liệu nhập | User input |
| Output | Dữ liệu xuất | Console output |

# Java Core

| English     | Vietnamese     | Example                                     | Note                        |
|-------------|----------------|---------------------------------------------|-----------------------------|
| Variable    | Biến           | `int age = 20;`                             | Lưu trữ dữ liệu             |
| Data type   | Kiểu dữ liệu   | `int`, `double`, `String`                   | Xác định kiểu dữ liệu       |
| Constant    | Hằng số        | `final double PI`                           | Không thay đổi giá trị      |
| Scanner     | Bộ đọc dữ liệu | `Scanner scanner = new Scanner(System.in);` | Nhập dữ liệu                |
| Input       | Dữ liệu nhập   | `Enter your age:`                           | Người dùng nhập             |
| Output      | Dữ liệu xuất   | `System.out.println()`                      | Hiển thị kết quả            |
| Class       | Lớp            | `public class Exercise01`                   | Đơn vị tổ chức chương trình |
| Method      | Phương thức    | `public static void main()`                 | Hàm trong Java              |
| Main method | Hàm main       | `main(String[] args)`                       | Điểm bắt đầu chương trình   |
| Object      | Đối tượng      | `Scanner scanner`                           | Thể hiện của một lớp        |

# Programming

| English        | Vietnamese  | Example                 | Note                |
|----------------|-------------|-------------------------|---------------------|
| Declaration    | Khai báo    | `int age;`              | Tạo biến            |
| Initialization | Khởi tạo    | `age = 20;`             | Gán giá trị ban đầu |
| Assignment     | Gán giá trị | `age = 25;`             | Cập nhật giá trị    |
| Expression     | Biểu thức   | `a + b * c`             | Sinh ra giá trị     |
| Statement      | Câu lệnh    | `System.out.println();` | Một lệnh hoàn chỉnh |
| Parameter      | Tham số     | `print(int number)`     | Trong khai báo hàm  |
| Argument       | Đối số      | `print(10)`             | Khi gọi hàm         |
| Return         | Trả về      | `return result;`        | Trả kết quả         |
| Exception      | Ngoại lệ    | `catch(Exception e)`    | Lỗi khi chạy        |
| Error          | Lỗi         | `Invalid input`         | Thông báo lỗi       |
